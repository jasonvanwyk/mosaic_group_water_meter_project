## Naming Conventions

### Files
- **Standard:** `lowercase-with-hyphens.md`
- **Numbered:** `NN-description.md` (two-digit prefix)
- **Dated:** `YYYY-MM-DD-description.md` (correspondence, site visits)
- **Versions:** `NN-description-vN.md` (when multiple versions needed)

### Folders
- **Format:** `lowercase-with-hyphens/`
- **Date-based:** `site-visit-DDMMYYYY/` (for photo folders)

### Reference Codes
- **Project Code:** MGW (Mosaic Group Water)
- **Correspondence:** `MGW-COR-YYYYMM-NNN`
- **Example:** `MGW-COR-202601-001`
